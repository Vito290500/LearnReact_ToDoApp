# Todo App - React

Una semplice applicazione Todo List realizzata in **React**, con funzionalità di aggiunta di nuovi todo tramite un modulo modale. Include anche la gestione della priorità, data e ora di ogni task.  
Ideale per imparare a lavorare con componenti, props, state e gestione di liste dinamiche in React.

---

## Features

- Aggiunta di nuovi todo tramite un form modale
- Visualizzazione di titolo, data, ora e priorità per ogni todo
- Checkbox per segnare un todo come completato (solo interfaccia, senza logica persistente)
- Pulsanti “Elimina” e “Modifica” pronti per futura implementazione
- Lista iniziale di todo di esempio (`TODO_TASK`)
- Stile moderno della lista con modale e input ben organizzati

---

## Tecnologie usate

- **React** (con Hooks `useState`)  
- **JavaScript / JSX**  
- **CSS** per stile della modale, lista e checkbox  
- Tooling: Vite
